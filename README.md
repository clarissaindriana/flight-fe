# Flight Booking Frontend

This repository contains a Vue.js-based frontend application for flight booking management. The application fetches data from a REST API backend available at [https://github.com/clarissaindriana/flight-be.git](https://github.com/clarissaindriana/flight-be.git).

## Project Overview

This is a personal project developed as an assignment for the course "Arsitektur dan Pemrograman Aplikasi Perusahaan" (Architecture and Programming of Enterprise Applications).
